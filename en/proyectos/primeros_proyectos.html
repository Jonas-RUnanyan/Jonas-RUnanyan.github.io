What I learned from this project:

  • Designing code skeletons to guide participants and encourage hands-on learning
  • Managing Git repositories effectively to prepare workshop materials and share resources
  • Improving my public speaking and technical presentation skills

This project was a Unity workshop we delivered for the Try It! conference at the Polytechnic University of Madrid in 2024. The idea was to explain the basics of procedural terrain generation using noise, as well as the implementation of a simple character controller to move around the environment. It was one of the most attended workshops of that year.

What I learned from this project:

  • Hands-on experience with Unity for tool and gameplay prototyping
  • Implementing object serialization and data handling with JSON
  • Working with a team in a structured setting, coordinating through regular meetings and following requirements

My first project at the MadHCI Lab consisted of developing a procedural room generator that allowed placing furniture and serializing them into JSON, functioning as an authoring tool. To achieve this, we implemented two modules: one to create and serialize rooms, and another to load them from a JSON file, thus keeping the logic separated.
